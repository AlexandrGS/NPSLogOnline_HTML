Приклад як запускати.
Розраховано на щомісячну ротацію логов NPS-сервера. файл логов повинен мати назву in2601.log, де 26 - це рік, а 01 - місяць

.\NPSLogOnline_HTML.ps1 -LogPath "c:\Windows\System32\LogFiles","\\192.169.0.1\c$\Windows\System32\LogFiles"
В цьому прикладі в двох папках шукаються файли з іменем inYYMM.log, де YY це останні ді цифри поточног року, а MM це дві цифри поточного місяця

.\NPSLogOnline_HTML.ps1 -LogPath "c:\Windows\System32\LogFiles","\\192.169.0.1\c$\Windows\System32\LogFiles" -CssStyleFile "http://192.169.0.1/vpnstat/NPSLogOnline_HTML_style.css"
тут додано файл з оформленням -CssStyleFile. Приклад такого файла дивись в цьому репозиторії

Також доступні параметри:

OutHTMLFile - файл куди буде зберігатися веб-сторінка з результатами (перелік поточних онлайн-користувачів)

OutLogFile - файл куди буде сохранятися лог. Потрібно тільки для отладки. Ротація логов не підтримуеться

IPGeoLoc_FileName - файл куди будуть зберігатися ІР адреси і відповідні їм гео-коордінати
 
 Перелік параметров не актуальний, дивитись початок powershell-файла.
